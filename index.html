TORONTO WEATHER

We created this web-application having in-mind all the people that can't seem to come up and decide on plans, including ourselves. With this project we aim to provide you with an event that's around you and save you some thinking and decision power.

This app uses your location to find events happening near you. The search gives events that are only happening after the moment of the search. You will only receive events that are active. In other words, events happening in the very near future. We provide you with the event title, address, as well as the event start (time and day).
